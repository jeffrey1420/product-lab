## Signal 1: MCP Ecosystem Hits Critical Mass

**97M downloads and counting.** The Model Context Protocol (MCP) — created by Anthropic and open-sourced November 2024 — is now the de facto standard for connecting AI agents to external tools. The March 2025 spec update (async tasks, enhanced sampling, elicitation) shipped the largest changes since launch.

Four protocols now define the agentic stack:
- **MCP** — agent to tools (APIs, DBs, files, browsers)
- **A2A** (Google) — agent to agent coordination (50+ launch partners, April 2025)
- **ACP** — agents for open commerce
- **UCP** — Google commerce layer

**Key insight for Alizé:** MCP is now a must-have integration point. Any managed AI service that can't connect to a client's tools via MCP will be at a disadvantage. Domo launched an MCP server March 25, 2026 specifically to connect enterprise data to external AI platforms. The trend: every SaaS tool will need an MCP server or be left out of agent workflows.

**13,000+ MCP servers launched on GitHub in 2025 alone.**

## Signal 2: Agentova — French Competitor Quietly at 2,000 Users

[Agentova](https://agentova.ai) is a French/Swiss SaaS platform (Cham, Switzerland) targeting entrepreneurs and small businesses with 8 specialized AI agents covering: customer service, prospecting, content creation, social media, follow-ups, admin, and inbound/outbound calls.

**The numbers:**
- 2,000+ active users in France
- 4.9/5 Trustpilot rating
- 10–20 hours/week time savings per business
- 82% average productivity gain
- Connects to 3,000+ tools (Instagram, WhatsApp, Notion, LinkedIn, Gmail, etc.)
- No-code deployment in days

**Client proof points:**
- Celesty (Fabien Dumas): -40,000 euros/month in operational costs, 2x team productivity
- Mandyben Formation (Benoît): +35% revenue via automated prospecting + follow-up sequences
- BullenSoi (Linda): calendar 100% filled through automated outbound prospecting

**What makes them different:** They call it the "AI Brain" — a contextual knowledge base with brand tone, products, rules, and processes. Agents don't just execute scripts; they reason within the brand's context.

**Alizé positioning gap:** Agentova is horizontal (entrepreneurs broadly). Alizé's angle — focused on operational IA for PME/ETI with governance, supervision, and structured deployment — is more targeted at companies that need accountability (industry-specific workflows, compliance, management reporting). Alizé should lean into the governance/governance story, not just automation.

## Signal 3: Domo AI Agent Builder + MCP Server (March 25, 2026)

Enterprise BI platform Domo announced a full AI orchestration framework at Domopalooza:
- **AI Agent Builder** — create conversational agents for specific use cases
- **AI Toolkits** — reusable packages of tools + data + context that define what an agent can do (role: financial analyst, ops manager)
- **Domo MCP Server** — exposes Toolkit capabilities to external AI platforms (Claude, Gemini, ChatGPT) via MCP

**Why it matters:** Even legacy enterprise SaaS is rushing to become "AI-native via MCP." Every data platform will need an MCP server. This validates the Alizé approach — enterprises are buying AI agents that connect to their existing data and workflows, not standalone chatbots.

## Signal 4: 2026 — The ROI Awakening (SaaStr)

Key thesis from SaaStr's analysis of the B2B AI moment:

> "2025: foundation year. Models actually good enough for production. Agents managing sub-agents. Early proof points. 2026: when it gets real."

The unlock timeline:
- **June 2024:** Claude 3.5 Sonnet — first unlock for AI coding
- **October 2024:** Major Sonnet upgrade (33% → 49% SWE-bench) — "actually good" unlock. Bolt.new hits $4M ARR in 30 days. Lovable hits $1M ARR in 2 weeks.
- **Early 2025:** "Vibe coding" explosion. YC Winter 2025 batch: 25% of codebases 95% AI-generated.
- **May 2025:** Claude Opus 4 — 72.5% SWE-bench, 7-hour autonomous sessions. Production-grade.
- **Mid-2025:** Agentic orchestration unlock — AI managing sub-agents. Full workflow automation becomes possible.

**For Alizé:** The "ROI awakening" is exactly Alizé's pitch. French PME don't want a chatbot demo — they want 40k euros/month in cost reduction or 35% revenue increase. Lead with the number, not the technology.

## Signal 5: AI SDR / Prospecting Tools Maturing Fast

AI agents are now orchestrating full sales cycles autonomously:
- Research → personalization → multi-channel sequencing → response handling → meeting scheduling
- Support: triage → route → resolve → escalate → follow-up → pattern analysis

This validates the cold-email-writer + mini-crm + web-scraping skill stack. Louis's team already has the core skills for a French AI prospecting tool. But Agentova is already doing this for the entrepreneur market.

**Alizé angle:** Position above pure prospecting tools. The value isn't the email — it's the supervised, governed, measurable workflow with clear accountability to management. French PME need someone to say "here's what the AI did last month, here's the ROI."

---

## Signal 6: Multi-Agent Orchestration — The New Default Architecture

Single AI doing one task → AI orchestrator + specialized sub-agents for each domain. This is the architectural pattern succeeding in production:
- Planning/orchestration agent
- Research agent (MCP: web search, DB)
- Writing agent (MCP: docs, CMS)
- Analysis agent (MCP: code execution)

**Implication for product building:** Any serious AI service offering in 2026 needs a multi-agent architecture. The Alizé managed service offering should be designed around orchestrating domain-specific agents with a governance layer on top.

## Actionable Insights for Alizé

1. **MCP-first integration:** Build Alizé's client connectors as MCP servers. Every client's tools (CRM, ERP, email, etc.) should be accessible via MCP. This is now table stakes.

2. **Prove the ROI from day one:** Agentova documents 10-20 hrs/week saved, 82% productivity gain. Alizé needs its own benchmarks per client vertical. "Managed AI agent" is a tough sell without numbers.

3. **French market is warming up:** Agentova proves French entrepreneurs are buying AI agent platforms now. The window for Alizé's positioning — managed IA with governance for PME/ETI — is open. But Agentova is expanding upward toward SMEs. Move fast.

4. **Multi-agent reference architecture:** Document a standard Alizé agent stack (orchestrator + domain agents + governance layer) as part of the pilot offering. Makes the service feel structured and enterprise-grade.

5. **Domain expertise > general automation:** The Agentova "AI Brain" concept (brand context, tone, rules) is powerful. Alizé pilots should invest in building a client's operational context before deploying agents.
