## Signal 1: Hermes Agent (NousResearch) — Self-Improving Agent That Grows With You

[NousResearch/hermes-agent](https://github.com/NousResearch/hermes-agent) is the most relevant new tool for Alizé's positioning. Key characteristics:

**Built-in learning loop**: Creates skills from experience, improves them during use, persists knowledge, searches past conversations. This is the "agent that gets better" thesis — exactly what a managed service operator needs.

**Multi-platform gateway**: Telegram, Discord, Slack, WhatsApp, Signal, and CLI — all from a single gateway process. For Alizé, this means: deliver agents to clients via whatever channel they already use, not a new portal.

**Runs on $5 VPS**: The docs explicitly say it runs on a $5 VPS, a GPU cluster, or serverless. This is the infrastructure economics Alizé needs — not enterprise GPU clusters, but commodity cloud.

**Model-agnostic**: Supports Nous Portal, OpenRouter (200+ models), z.ai/GLM, Kimi/Moonshot, MiniMax, OpenAI, or custom endpoints. Switch with one command. No vendor lock-in.

**Closed learning loop**: Agent-curated memory with periodic nudges, autonomous skill creation after complex tasks, FTS5 session search with LLM summarization, Honcho dialectic user modeling.

**Alizé implication**: Hermes validates the managed agent service model at the right price point. Alizé could use Hermes as a reference architecture or even a building block for client deployments — particularly for smaller PME clients who need Telegram or WhatsApp delivery. The model-agnostic design using z.ai/GLM and MiniMax is directly compatible with Alizé's existing setup.

## Signal 2: TaxHacker — Self-Hosted AI Accounting (The Freemium Threat)

[vas3k/TaxHacker](https://github.com/vas3k/TaxHacker) is a self-hosted AI accounting app for freelancers and small businesses. It:

- Accepts photos of receipts/invoices → AI extracts date, amount, vendor, line items, taxes
- Auto-categorizes transactions with custom prompts
- Multi-project support, currency conversion (including crypto), Excel export
- Runs fully self-hosted

**Alizé implication**: This is exactly the category of tool that could become a freemium product targeting French artisans and micro-enterprises (the "artisans" segment Louis's prospecting business was targeting). If TaxHacker or a similar tool adds French accounting standards (PCG, TVA rules, French invoicing requirements), it could become a zero-barrier entry point for the French micro-enterprise market. Alizé should watch this space: either partner with/complement such tools, or ensure its own workflow integration layer adds enough value to justify the managed service price.

## Signal 4: Superpowers — Agentic Skills Framework for Software Dev

[obra/superpowers](https://github.com/obra/superpowers) is an agentic skills framework + software development methodology for coding agents. Key idea: instead of agents jumping straight into code, they step back, derive specs, show them to the user in digestible chunks, then use subagent-driven development.

- Available via official Claude plugin marketplace: `/plugin install superpowers@claude-plugins-official`
- Emphasizes TDD, YAGNI, DRY
- Agents can work autonomously for hours without deviating from plan

**Alizé implication**: The "spec-first, human-in-the-loop" design pattern is directly applicable to Alizé's client onboarding workflow. Before deploying agents to a new client, Alizé should derive a spec (workflow definition) and get client sign-off before automation runs. This reduces scope creep and increases trust.

## Signal 5: France AI Strategy — €109B Announced, But Where's the PME Layer?

The Élysée announced €109 billion in AI infrastructure investments at the AI Summit (February 2025). France's advantages cited: decarbonized electricity, high-voltage grid, suitable data center sites, streamlined procedures.

**However**: This is almost entirely infrastructure investment (data centers, GPU clusters, cloud). The French government's AI strategy has historically been strong on research and infrastructure, weak on PME/TPE adoption. The recent AI Summit announcements are hyperscaler-focused.

**Alizé implication**: There's a gap between France's AI infrastructure investments and actual PME adoption. Alizé's managed service model — "we handle everything, you get results" — is precisely designed to fill that adoption gap. This is an opportunity to frame Alizé as the missing layer between government AI infrastructure and actual French PME workflow transformation.

**Action**: Investigate whether any French government programs (Bpifrance, France 2030, regional aid) could co-fund AI adoption for PME. Some relevant programs:
- [France 2030](https://www.gouvernement.fr/france-2030) — includes AI adoption for businesses
- [Bpifrance](https://www.bpifrance.fr) — PME innovation financing
- DINUM (Direction Interministérielle du Numérique) — public sector AI programs
