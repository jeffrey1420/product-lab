## Signal 1: Cloudflare Launches EmDash — WordPress Rebuilt with AI Agents

Cloudflare published EmDash (emdash-cms.com), described as "the spiritual successor to WordPress." Key facts:

- **Built in 2 months using AI coding agents** — Cloudflare's team rebuilt WordPress from scratch using AI agents. This is a direct proof point that AI-assisted dev has crossed from demos into production-grade software.
- **TypeScript + Astro + serverless** — Fully MIT licensed, runs on Cloudflare Workers or any Node.js server
- **Plugin security solved** — Plugins run in isolated Cloudflare Workers Dynamic Workers with explicit capability manifests. 96% of WordPress security issues come from plugins; EmDash's model makes plugins inherently sandboxed.
- **Available now** — v0.1.0 preview on GitHub (emdash-cms/emdash), deploy to Cloudflare or self-host

**For Kuroba (web agency):** EmDash is a warning shot. It demonstrates that AI coding agents can build a full WordPress-replacement CMS in 2 months. If Cloudflare can do this, future clients will expect sites built faster and cheaper. Kuroba's positioning should lean even harder into human design + strategy + local presence — not code delivery.

**For Alizé:** EmDash's capability-manifest model for plugins is architecturally similar to how Alizé agents should declare and scope their permissions. Worth studying as a reference pattern.

## Signal 2: 1-Bit LLMs Hit Commercial Viability — Inference Costs Set to Drop

PrismML (prismml.com) launched "1-Bit Bonsai" — commercially viable 1-bit LLMs — on Hacker News (402 points, April 1).

- **What it is:** LLMs where model weights are compressed to 1-bit (or near-1-bit) precision. Similar to the BitNet b1.58 approach from Microsoft Research.
- **Why it matters:** 1-bit LLMs can run inference at a fraction of the compute cost of full-precision models. Potentially 10-100x cheaper at scale.
- **Commercial viability claim:** PrismML is the first to ship this as a commercially available product (not just a research paper)
- **Alignment:** Alizé's cost structure depends heavily on inference costs. 1-bit models could make vertical AI agent services significantly more profitable or enable lower pricing.

**For Alizé:** Monitor PrismML's pricing and performance benchmarks closely. If 1-bit models achieve reasonable quality on French business tasks (invoice processing, email drafting, CRM updates), Alizé could pass savings to clients or improve margins. This is a potential structural advantage for the pricing model.

## Key Takeaways for Alizé

**1. Inference costs are falling — plan for it**
1-bit LLMs (PrismML) and cost-effective models (StepFun 3.5 Flash) validate that the cost-per-task curve is steeply declining. Alizé's model should assume falling inference costs over the next 12-18 months — don't anchor pricing on today's GPT-4 costs.

**2. EmDash validates AI-assisted dev, signals more competition**
If Cloudflare can rebuild WordPress in 2 months with AI agents, the dev velocity gap widens. Kuroba should lean into client intimacy and design thinking, not compete on speed.

**3. Protected's bootstrapped path is a template**
€20M+ ARR, profitable, bootstrapped. No VC. Partner distribution. This is exactly the playbook Alizé could follow — start with 3-5 pilot clients, build a partner channel (accountants, consultants), grow lean.

**4. The French health AI reimbursement path is opening**
Joe/Ludocare getting HAS reimbursement approval signals regulatory willingness. For Alizé, health-adjacent verticals (pharma, medical billing, insurance) could become viable.

**5. Alizé's immediate action items:**
- Buy the domain (alize.ai or alize.fr — this is week 3 of the project, still not done)
- Build MCP connector for Sage (technical moat, French accounting integration)
- Draft landing page positioning (vertical specialization over generic agents)
