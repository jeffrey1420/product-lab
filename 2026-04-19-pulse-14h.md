## Signal 1: Opus 4.7 System Prompt — Agent Autonomy Up, Verbosity Down

**Source:** Simon Willison (simonwillison.net, April 18, 2026), HN item #47823270 (58 pts, 3h ago)

Anthropic published the system prompt diff between Opus 4.6 (Feb 5, 2026) and Opus 4.7 (April 16, 2026). Key changes:

**Agentic autonomy explicitly rewarded:**
- Claude now "calls the tool to try and solve the ambiguity before asking the person" — tool use preferred over clarification questions
- "tool_search" mechanism added: Claude checks if a relevant tool exists before saying "I don't have access to X"
- Once Claude starts a task, "sees it through to a complete answer rather than stopping partway"

**Multi-app agent expansion:**
- Claude in Chrome (browsing agent), Claude in Excel (spreadsheet agent), Claude in PowerPoint (slides agent) — all now in the system prompt
- Claude Cowork can use all of these as tools — desktop agent orchestration layer is now official behavior

**Persona changes:**
- Removes "avoid saying genuinely, honestly, or straightforward" — model is less restricted in natural language
- Removes asterisk emote restriction — more expressive
- New guardrails: disordered eating topics, screenshot jailbreak (simple yes/no on contested issues)

**Why this matters for Alizé:**

Opus 4.7's explicit push toward autonomous tool use + multi-app agents is exactly the trend creating Alizé's market. As Claude becomes more agentic out of the box — autonomously browsing, using spreadsheets, creating slides — enterprises need governance infrastructure proportional to that autonomy. More agentic capability = more audit surface = more Alizé opportunity.

The PowerPoint agent is particularly notable: if AI agents start creating client-facing presentations autonomously, French SMEs face the same securities fraud / misrepresentation risk described in the April 13 Bloomberg Law signal. Audit trails for AI-generated documents become a legal requirement.

The "sees it through to completion" directive also means fewer graceful handoffs back to humans — which is the exact gap Alizé's human-in-the-loop layer fills.
