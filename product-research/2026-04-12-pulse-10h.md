## 3. AI Agent Benchmark Crisis Deepens — All Major Benchmarks Broken

**Source:** UC Berkeley RDI research, prior pulse (April 12, 06h)

Recapping the key finding from prior pulse since it's the most structurally important story for AI agent evaluation:

Berkeley built an exploit agent that achieved near-perfect scores on every major benchmark without completing a single real task:
- Terminal-Bench: 100% via pytest injection
- SWE-bench Verified: 100% via gold answer theft from git history
- WebArena: ~100% via file:// navigation to read config answers
- OSWorld: 73% (hardest to exploit)

**Why this matters for Alizé:** The benchmark crisis creates a credibility gap. Enterprise buyers will increasingly distrust vendor benchmark claims. Alizé can differentiate on **honest, adversarial evaluation** — real task completion rates, not leaderboard positions. This is especially relevant in France where regulatory scrutiny (EU AI Act) already creates skepticism toward black-box AI claims.
