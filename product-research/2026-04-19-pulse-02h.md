## Signal 2: Claude Design → Claude Code — A Real Production Workflow Emerges

**Source:** Hacker News, Sam Henri Gold blog (April 18, 2026)

A user on Hacker News documented a **real production workflow** using Anthropic's tools sequentially:

1. **Claude Design** → generate design options for a new footer section
2. Iterate on the output
3. **Claude Code** → build the design (with integrated feature from Claude Design)
4. Deploy to production

The result is live at tediware.com — a footer section with "Origin story" on the left and a signup panel on the right.

**Key observations:**
- This is not a demo or toy — it's an actual iPaaS startup using the Anthropic stack for production work
- The Claude Design → Claude Code integration ("that integrated feature is very cool") is a direct code handoff
- The HN discussion also surfaced the **homogenization problem**: Claude Design outputs look similar across users, just like AI text/code/imagery has a homogeneous feel. This is a real limitation for brand-differentiated work.

**The Figma problem (from the blog post):**

The blog post makes a sharp observation: **Figma's locked-down, undocumented format excluded it from LLM training data.** Code was trained on; Figma primitives weren't. As designers can now write code directly (via Claude Design and similar tools), the source of truth naturally migrates back to code. Figma's baroque token/component/variable system — described in the post as "a lossy approximation" of code — becomes harder to justify.

**Why this matters for Alizé:**

1. **The agentic stack is stacking up.** Claude Design (design) + Claude Code (build) + managed infrastructure = a complete design-to-deployment pipeline. Alizé's opportunity: governance layer above this stack for enterprise deployments.

2. **Figma is vulnerable.** Their enterprise customers (design teams) are being courted by Anthropic's tools. Figma's response (Figma Make) still treats the design file as canonical — the opposite bet from Claude Design. If Figma loses the "source of truth" war, enterprise design governance (a potential Alizé adjacencies) becomes a market.

3. **Small teams can now ship faster.** The iPaaS startup example: two tools, one deployment. The gap between idea and production is shrinking. This accelerates the overall AI agent deployment pace — and the need for governance.
