# ts.news Daily — March 23, 2026

## Topics Researched

- **TypeScript 7 Native Preview (Project Corsa):** Microsoft ported the TypeScript compiler and language service to Go. Benchmarks show VS Code compiling in 7.5s vs 77.8s previously (~10x faster). Project load times down ~8x. TypeScript 6.0 is the last JS-based major release; 7 marks the native era. Node.js also shipped native type stripping in 22.18.0, enabling direct `.ts` execution without transpilation.

- **AI Coding Tool Showdown 2026:** Claude Code (Anthropic) went from zero to #1 in 8 months, reaching 46% "most loved" vs Cursor 19% and Copilot 9%. Claude Code wins on large refactors and agentic workflows (CLI-based, Claude Opus 4.6 model, SWE-bench 74.4%). Cursor wins on day-to-day frontend work and quick edits ($20/mo vs $100-300/mo for Claude Code). Copilot still strongest on GitHub ecosystem integration.

- **Node.js vs Bun vs Deno 2026:** Bun continues to gain traction as a drop-in Node replacement with better performance. Deno positions on security. Node.js 22/23 shipped native TypeScript type stripping by default — a major shift reducing the need for transpilation in dev/test environments.

- **Vue vs React 2026:** Both frameworks moving toward "hybrid rendering" — surgically choosing per-component how HTML is delivered. Nuxt 4 and Next.js continue converging on similar mental models. Vue 3.5's rewritten reactivity (56% less memory, 10x faster on large arrays) remains a significant recent improvement. React 19.2 and Angular 21 also released recently.
